# Release Notes v0.14.0

**Release Date:** 2026-07-19

This release makes `search-prs` output stdout-first with selectable
formats, refreshes dependencies, and standardizes repository shields.

## Highlights

- **search-prs stdout default and multi-format output**: results print to
  stdout by default with selectable output formats, making the command
  pipeline-friendly (previous behavior wrote files such as xlsx exports;
  file output remains available via format selection)

## Documentation

- `search-prs` CLI documentation updated for the stdout default
- README shields standardized, with a dev-guide badge pointing at the
  MkDocs site

## Dependencies

- `go-isatty` v0.0.23, `excelize` v2.11.0
- GitHub Actions `setup-go` bumped from 5 to 7

## Housekeeping

- `.envrc` and local xlsx exports are now gitignored

## Installation

```bash
go get github.com/grokify/gogithub@v0.14.0
```

[CHANGELOG.md](https://github.com/grokify/gogithub/blob/main/CHANGELOG.md)
